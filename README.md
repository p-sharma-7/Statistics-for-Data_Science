   - Summary stats, outlier handling, feature analysis

4. **Probability Theory**

       - Stochastic gradient decent
       - Mini - batch Gradient descent
   - Gradient Descent 3D

   - DBSCAN

   - EDA and analysis on datasets like Titanic, Iris, and more
